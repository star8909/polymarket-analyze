📊 실거래 점수 (100점 만점) — 폴리마켓 기준

1. Edge (35점) = +EV 5%+ 만점, 3%+ 25점, 1%+ 15점
2. 샘플 수 (15점) = 100+ 마켓 만점 (calibration 신뢰도)
3. 운용 친화도 (20점) = 자동화 가능(10) + Polygon API 안정(5) + Kelly sizing(5)
4. 리스크 robustness (20점) = UMA 분쟁 대비 + 카테고리 분산
5. Out-of-sample (10점) = 학습 기간 외 검증 통과
● 80+ 베팅 시도 가능 (소액 paper trade) ● 60-79 추가 검증 필요 ● < 60 베팅 불가
✅ 도메인 지식 불필요 — 종가/거래량 패턴만으로 +EV. 비트코인 481 룰 발견 시스템과 같은 철학.
✅ Calibration이 가장 강력한 무기 — 종료 마켓 결과 = 정답. 시장 가격 ≠ 진짜 확률 영역 자동 발견.
✅ Sum-to-1 차익은 순수 산수 — multi-outcome 마켓 가격 합 위반 = 무위험 차익.
⚠️ UMA 오라클 분쟁 risk — 마켓 무효화 가능성. 꼬리 위험 항상 인지.
⚠️ 결과 누출 마켓 회피 — 1주일 미만 남은 마켓 false signal 多. 7일+ 필터 필수.
iter 전략 핵심 로직 Edge Sharpe (연환산) 승률 N Kelly 연수익률 추정 위험도 점수 판정 비고

📋 분석 전략 (iter 진행 순서)

Phase 1 — Calibration (iter02): 가격 → 실제 결과 buckets. 5센트 롱샷, 95센트 favorite 편향 측정.

Phase 2 — Sum-to-1 (iter03): multi-outcome 가격 합 위반 = 무위험 차익.

Phase 3 — Time-series 분포 (iter04+): Beta/Logit/GARCH로 가격 ±2σ 극단 → mean reversion 베팅.

Phase 4 — Cross-market 일관성 (iter05+): 조건부 마켓 모순 (예: 후보 가격 > 당선 가격) 차익.

Phase 5 — Kelly Sizing (iter06+): Fractional Kelly (0.25x) 사이즈 최적화.

🛡️ 베팅 안전 규칙

1. Paper trading 4주 이상 — 백테스트 vs 실거래 갭 확인

2. 자본 5% 이하만 — UMA risk + 외환 손실 + 가스

3. Fractional Kelly 0.25x — 추정 오차 흡수

4. 마켓당 max $100 — 단일 마켓 위험 캡

5. 카테고리 분산 — 정치/스포츠/크립토 한 곳 몰빵 X

6. 자동화 필수 — 24h 마켓에서 수동 모니터링 X
